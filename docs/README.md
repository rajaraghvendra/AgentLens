# Dashboard Screenshots

This directory contains placeholder references for dashboard screenshots.

## Required Screenshots

To complete the documentation, capture and save these screenshots:

### 1. dashboard.png
Main dashboard view showing:
- Total Cost, Sessions, Tokens, Cache Efficiency, Providers metrics
- Daily cost chart
- Projects breakdown
- Activities distribution
- Warning alerts (if any)

### 2. optimize.png
Optimize tab showing:
- Health Grade card (large letter grade)
- Findings list with severity badges
- Optimization insights

### 3. compare.png
Compare tab showing:
- Model comparison table with columns: Model, Sessions, Total Cost, Cost %, Tokens, Avg Cost/Session

## How to Capture

1. Start the dashboard: `npm run dashboard`
2. Navigate to http://localhost:3000
3. Use your OS screenshot tool (Cmd+Shift+4 on macOS)
4. Save as PNG files in this directory

## TUI Screenshots (Optional)

If you want to document the TUI:
1. Run: `agentlens tui`
2. Capture the terminal window
3. Save as docs/tui.png
